# 文件自动主题分类（无监督）

简介
- 这是一个基于语义向量与 K-Means 的无监督文档主题分析/标签系统。
- 输入：PDF / DOCX / TXT 文档。
- 输出：文档级主题标签（基于文档中段落聚类的最大主题）以及主题分布与代表段落示例。

关键点（设计说明）
- 这是无监督方法：没有预定义类别，系统通过对段落进行聚类来发现文档中的主题。
- 文档级标签定义为“文档中样本数最多的主题（最大簇）”，同时返回全部主题的分布与示例段落，便于人工判断或后续处理。
- 系统同时提供肘部法则（Elbow Method）与轮廓系数（Silhouette）来辅助选择聚类数 K。

文件结构
- `app.py`：FastAPI 服务，接收文件并返回主题结果。
- `cluster_kmeans.py`：Embedding 封装、KMeans 聚类、肘部法则与自动生成聚类名称的逻辑。
- `static/stopwords.txt`：分词停用词表。
- 其它模型文件夹（如 `bge-small-zh-v1.5/`）包含语义模型权重（如使用本地 sentence-transformers 模型）。
